Introduction to Website Development

About This Website

This informational website was created as part of an Ivy Tech course project to demonstrate basic website development concepts and hosting using GitHub Pages.

Purpose of the Site

The purpose of this site is to present organized information using clear writing, semantic HTML structure, and simple design principles. Accessibility and readability are considered throughout the development process.

Technologies Used

  • HTML for page structure
  • GitHub for version control
  • GitHub Pages for web hosting

Accessibility Considerations

This website uses semantic HTML elements, proper heading structure, and readable text to support accessibility and usability for all users.
